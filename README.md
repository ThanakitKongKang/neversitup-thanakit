# Installation
- Change **.env.example** to **.env**
- Correct DB_USER & DB_PASS in **.env**
- Create SQL schema named **neversitup_thanakit**

then
```bash
npm install
```
then
```bash
npm start
```

  Base URL [http://localhost:8080](http://localhost:8080)
